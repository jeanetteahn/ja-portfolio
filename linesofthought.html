Lines of Thought

A digital gallery bridging art history and web design through curated storytelling and interactive experiences.

Role Web Designer & Developer
Project Type Personal/Academic
Timeline March - April 2025
Tools HTML, CSS, JavaScript

Project Brief

Lines of Thought is a digital gallery created as a personal project for my web design class. It curates artworks, sculptures, and architectural pieces in a virtual walkthrough experience, with each work paired with a museum-style label combining context and personal reflection.

As an homage to my Art History undergraduate background, the project explores how digital design can translate the contemplative atmosphere of a museum into an accessible online format‚Äîbridging my past academic training with my current focus in UX design.

Process

Research & Curation

  • Outlined the collection, selected works, and wrote interpretive labels.
  • Researched historical/artistic context to ground each piece in storytelling.
  • Prepped and optimized images for web display.

Curatorial research and artwork selection process

Design & Development

  • Built the gallery using HTML/CSS after finding React unnecessarily complex for the scope.
  • Created a clean site structure to support JavaScript lightbox functionality.
  • Iterated on responsive layouts using flexbox and media queries for consistency across devices.

Site structure and responsive layout development

Refinement & Accessibility

  • Debugged lightbox duplication caused by unwrapped HTML elements.
  • Resized logo and navigation for improved hierarchy and readability.
  • Passed accessibility validations and standardized formatting across pages.
  • Added a typewriting animation on the homepage to reinforce the theme of "lines of thought," refining cursor blinking to avoid distraction.

Accessibility testing and final refinements
